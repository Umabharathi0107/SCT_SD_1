## 📌 Task 01: Temperature Conversion Program

### 📖 Project Description
The first task was to develop a functional program that converts temperatures between three primary scales: **Celsius**, **Fahrenheit**, and **Kelvin**. This project demonstrates fundamental programming concepts such as user input handling, mathematical logic, and conditional flow.

### 🛠️ Key Features
* **Multi-Unit Support:** Convert from any scale to the other two simultaneously.
* **Input Validation:** Handles non-numeric inputs gracefully using `try-except` blocks.
* **Scientific Accuracy:** Implements precise conversion formulas (including the $273.15$ constant for Kelvin).
* **Clean UI:** Simple, interactive command-line interface with formatted outputs.

### 🧪 Formulas Applied
| From | To Celsius | To Fahrenheit | To Kelvin |
| :--- | :--- | :--- | :--- |
| **Celsius** | - | $C \times \frac{9}{5} + 32$ | $C + 273.15$ |
| **Fahrenheit** | $(F - 32) \times \frac{5}{9}$ | - | $(F - 32) \times \frac{5}{9} + 273.15$ |
| **Kelvin** | $K - 273.15$ | $(K - 273.15) \times \frac{9}{5} + 32$ | - |
